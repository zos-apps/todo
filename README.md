## About

Simple and elegant task management application.

## Features

- Task lists
- Due dates
- Priority
- Categories
- Sync

## Installation

### From App Store
Open the **App Store** in zOS and search for "Reminders".

### Via npm
```bash
npm install @zos-apps/todo
```

## Usage

- Add tasks with +
- Check to complete
- Swipe to delete

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New task |
| `Enter` | Save |
| `Delete` | Remove |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
